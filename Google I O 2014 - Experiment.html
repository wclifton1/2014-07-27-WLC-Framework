What it is

This year's Google I/O web experiment explores several breakthroughs in science and technology. We're inspired by the power of science and technology that allows us to unlock discoveries in the universe as well as the everyday. Join us for Google I/O 2014 as we talk about our latest innovations in platforms, apps and APIs for developers.

How to use it

Tap it. Spin it. Swipe it. Click it. Unlock each chapter to arrive at the next.

How it was built

The experiment was built only using CSS 3D rather than WebGL so it could run on modern browsers and devices. We used SVG vector graphics so we could scale infinitely and have high fidelity images. Web Audio API brings audio input and output to the experience, while an accelerometer provides physical input.
